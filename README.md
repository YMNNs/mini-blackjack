# Mini Blackjack

用 React 编写的二十一点游戏。支持分牌、双倍下注、保险等玩法。

[在线 DEMO](https://ymnns.github.io/mini-blackjack/)

## 游戏规则

[查看游戏规则](rules.md)

## 本地开发

### 系统需求

Node.js >= 20

### 安装依赖

```shell
$ npm install
```

### 启动开发服务器

```shell
$ npm dev
```

### 测试

```shell
$ npm test
```

### 构建

```shell
$ npm build
```
